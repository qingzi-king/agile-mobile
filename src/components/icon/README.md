## 图标的命名规范

为每个图标赋予了语义化的命名，命名规则如下:

- 采用大驼峰形式，如：AbnormalOutline

- 命名顺序：`[icon名][形状可选][分类]`。

## 如何使用

```tsx

import { Icon } from 'agile-mobile'

const { BookFill } = Icon

<BookFill />

```
